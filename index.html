01

Syntax basics

C is a compiled language — you write code, a compiler translates it into machine instructions, and then it runs. Unlike Python, C demands you be explicit about everything. That strictness is what makes it fast and what makes beginners cry.

Every C file ends in .c. At minimum, your program needs exactly this:

// include the standard input/output library #include <stdio.h> // main() is the entry point — where execution begins int main() { // your code goes here return 0; // 0 = success }
ℹ
Every statement ends with ;. Forgetting it is the most common beginner mistake. The compiler will yell at you, and rightly so.

Comments are ignored by the compiler. Write them for future-you, who will have absolutely no memory of what past-you was doing.

// single-line comment /* multi-line comment: write as much as you want in here */
ℹ
You can only have one main() per project. It's the program's starting gun. More than one and the compiler won't know where to begin.

A variable declared inside a scope does not exist outside of it. It's destroyed when the scope closes. Try to use it outside and the compiler refuses.
03

Variables & data types

A variable is a named box in memory that holds a value. C is statically typed — you must declare what kind of value the box holds before using it. No guessing. The compiler needs to know.

TypeStoresExampleNotes
intwhole numbersint x = 42;No decimals — assigning 12.9 gives you 12
floatdecimals (~7 digits)float f = 3.14;Good for most decimal needs
doubledecimals (~15 digits)double d = 3.14159265;More precise, preferred for math
charsingle characterchar c = 'A';Stored internally as integer 0–255
int age = 21; float gpa = 3.7; double pi = 3.14159265358979; char grade = 'A'; // declare multiple at once int x = 1, y = 2, z = 3;
ℹ
Dividing 5/2 gives 2 (integer division). Dividing 5.0/2 gives 2.5. Always check your types when doing math.

Printf & scanf

printf prints to the terminal. scanf reads from the user. Both use format specifiers as typed placeholders inside the string.

Printf builder — edit the values and watch the output update
name = score = gpa =
Terminal output
%d→integer (int)
%f→float or double — use %.2f for 2 decimal places
%c→single character (char)
%s→string (char array)
\n→newline — printf won't add one automatically
// scanf reads INTO variable addresses using & int n; char c; scanf("%d", &n); // reads an integer into n scanf(" %c", &c); // space before %c skips leftover newline
⚠
Always put a space before %c in scanf: write " %c". Without it, scanf picks up the newline from the previous Enter key and your variable gets garbage.

For checking one variable against a specific list of values, switch-case is cleaner than a long chain of else-ifs:

switch (day) { case 1: printf("Monday\n"); break; case 2: printf("Tuesday\n"); break; case 6: /* falls through */ case 7: printf("Weekend!\n"); break; default: printf("Unknown\n"); }
⚠
Always write break; at the end of each case. Without it, execution falls through to the next case — sometimes useful (weekend above), almost always a bug.

// FOR — use when you know how many times upfront for (int i = 0; i < 5; i++) { printf("%d\n", i); } // WHILE — use when condition-driven, count unknown int n = 5; while (n > 0) { printf("%d\n", n); n--; } // DO-WHILE — runs at least once, checks condition after int n = 1; do { printf("%d\n", n); n++; } while (n <= 5);
ℹ
break exits the loop immediately. continue skips the rest of the current iteration and jumps to the next one. break also works inside switch-case.
08

Functions

A function is a reusable named block of code. Write it once, call it anywhere. This is the single most important habit in programming — write it once, use it many times, fix it in one place.

int   sumTwo  (int x, int y) { ... }
return type  what the function sends back function name  how you call it parameters  input values it needs
// prototype — tells the compiler the function exists before main int sumTwo(int, int); void greet(); int main() { int result = sumTwo(3, 4); // result = 7 greet(); return 0; } int sumTwo(int x, int y) { return x + y; } void greet() { // void = returns nothing printf("Hello!\n"); }
ℹ
Function parameters are copies. Changing x inside a function does not affect the original variable. To modify the original, you need pointers — that's the next chapter.

Arrays
